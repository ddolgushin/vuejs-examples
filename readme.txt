Предварительные действия:
  * установить Visual Studio Code (https://code.visualstudio.com/) + расширения:
    - C#;
    - Vue.js Extension Pack;
    - Debugger for Chrome;
  * установить NodeJS;
  * установить PostgreSQL 9.6+.

Запуск проекта "todo":
  * выполнить команду "npm install" в директории проекта для установки необходимых пакетов;
  * запустить Visual Studio Code и открыть папку проекта в редакторе;
  * запустить отладку с помощью клавиши F5.

Запуск проекта "marathon":
  * выполнить команду "dotnet restore" в корне проекта;
  * выполнить команду "npm install" в "marathon/ClientApp/" для установки необходимых пакетов;
  * запустить Visual Studio Code и открыть рабочую область "marathon/vue_marathon.code-workspace";
  * запустить клиентскую часть:
    - в редакторе кода открыть окно терминала (Ctrl+`);
    - создать новый терминал с помощью кнопки "+", в выпадающем списке выбрать "ClientApp";
    - в созданном терминале выполнить команду "npm run dev" для запуска Веб-сервера;
    - с помощью клавиши F5 запустить отладчик для клиента. На этом шаге должна отобразиться стартовая страница приложения;
  * запустить серверную часть с помощью клавиши "Запуск" в левой нижней части окна редактора, в выпадающем списке выбрать "vuejs:chrome";
  * заполнить базу начальными данными с помощью обращения по адресу: http://localhost:5678/api/admin/restore. В результате должна отобразиться пустая страница. В этой операции происходит обращение к соответствующему контроллеру, который создаст необходимые для начала работы объекты, в том числе учётную запись пользователя admin;
  * начать работу с приложением. Для авторизации можно использовать учётную запись "admin/admin".
